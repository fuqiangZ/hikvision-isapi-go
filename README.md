# 海康摄像头 isapi 客户端

语言：GOLANG

验证环境：Linux 、Mac、Windows

## 功能简介

封装了部分海康摄像头的 ISAPI 接口能力，目前重点在摄像头布防。

1. 封装了 ISPAPI 接口
2. 结构化转化：将布防长连接返回的数据，结构化为可用的数据；

## SDK 能力

| 接口名称 | 接口 | 功能描述                                       | 状态 |
| -------- | ---- | ---------------------------------------------- | ---- |
| 能力     |      |                                                |      |
| 布防     |      | 摄像头一级布防（ISAPI 原生接口不支持二级布防） |      |
